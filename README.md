# DSS — рабочее пространство

Здесь хранятся файлы и папки для совместной работы.

## Структура папок

- **Маша/** — личная папка. Внутри лежит `Маша.xlsx` (пустая таблица Excel).
- **Документы/** — сюда складывай вордовские файлы (`.docx`), PDF и прочие документы.
- **Таблицы/** — сюда складывай Excel-таблицы (`.xlsx`).

> Папки `Документы/` и `Таблицы/` пока пустые. В каждой лежит технический файл
> `.gitkeep` — он нужен только для того, чтобы Git «видел» пустую папку.
> Когда положишь туда настоящие файлы, `.gitkeep` можно удалить.

## Как добавлять файлы

Excel (`.xlsx`), Word (`.docx`), PDF, картинки — всё это можно загружать.
Через сайт GitHub: открой нужную папку → кнопка **Add file** → **Upload files**.
